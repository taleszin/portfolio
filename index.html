Sobre Mim

Olá! Sou estudante de Ciência da Computação na UECE, mas minha verdadeira sala de aula tem sido o desafio de construir software do zero. O que me fascina é o poder da arquitetura de sistemas para traduzir a complexidade do mundo em soluções digitais elegantes. Seja otimizando uma consulta em SQL ou projetando uma nova feature, busco a união entre performance técnica e uma experiência de usuário intuitiva.

  • PHP

    PHP - A Base de Tudo

    Comecei com PHP em 2021 criando meu primeiro CRUD na universidade. Desde então, construí APIs robustas, sistemas MVC complexos e plataformas SaaS como o MedLeap AI. É minha linguagem principal para arquitetar soluções escaláveis.

  • JavaScript (ES6+)

    JavaScript - Criando Experiências

    Mergulhei no JS moderno criando visualizações interativas para o ShotMap Basketball. Domino tanto vanilla JS quanto frameworks, focando em performance e UX fluida. Uso ES6+ para código limpo e manutenível.

  • MySQL

    MySQL - Dados que Fazem Sentido

    Aprendi otimização de queries trabalhando com milhares de registros no Portal da História do Ceará. Implementei índices Full-Text e modelagem relacional complexa para alta performance em buscas.

  • LLM & IA

    IA - O Futuro é Agora

    Integrei LLMs via API no MedLeap AI, criando simulações clínicas inteligentes. Trabalho com prompt engineering e fine-tuning para soluções educacionais inovadoras que realmente agregam valor.

  • AWS Cloud

    AWS - Escalabilidade na Nuvem

    Deploy e gerenciamento de aplicações em produção usando EC2, RDS e S3. Aprendi na prática configurando infraestrutura para projetos reais que precisavam suportar múltiplos usuários.

  • Figma (UI/UX)

    Design - Antes do Código, a Experiência

    Uso Figma desde 2022 para prototipar interfaces antes de codificar. Minha experiência na apLIS me ensinou que design thinking é fundamental para criar produtos que os usuários realmente querem usar.

  • Arquitetura de Sistemas

    Arquitetura - Pensando em Grande Escala

    Projetei a arquitetura do MedLeap AI pensando em escalabilidade e manutenibilidade. Aplico padrões como MVC, Repository e princípios SOLID para código que dura e evolui.

  • APIs RESTful

    APIs - Conectando Mundos

    Construí APIs desde endpoints simples até integrações complexas com IA. Minha API para o Portal da História serve milhares de consultas diárias com performance e segurança impecáveis.

  • Node.js

    Node.js - JavaScript Everywhere

    Uso Node.js para automações, ferramentas de build e microserviços. A versatilidade do JavaScript no backend me permite criar soluções full-stack coesas e eficientes.

  • Three.js

    3D - Experiências Imersivas

    Criei experiências 3D interativas no projeto Valentine's Day. Three.js me permite transformar ideias criativas em realidade, combinando arte e tecnologia para storytelling único.

  • POO

    POO - Código que Faz Sentido

    Programação orientada a objetos é meu fundamento desde a faculdade. Aplico conceitos como herança, polimorfismo e encapsulamento para criar código reutilizável e manutenível.
